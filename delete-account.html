계정 및 데이터 삭제

나무보카 앱에서 계정과 모든 개인 데이터를 삭제하는 방법을 안내합니다.

나무보카 앱에서 직접 계정과 모든 데이터를 삭제할 수 있습니다.

단계별 삭제 방법:

  1. 나무보카 앱을 실행합니다
  2. 화면 하단의 '설정' 탭을 선택합니다
  3. '계정 관리' 메뉴를 선택합니다
  4. '계정 및 데이터 삭제' 옵션을 선택합니다
  5. 삭제될 데이터 목록을 확인합니다
  6. '삭제 확인' 버튼을 선택하여 최종 삭제를 진행합니다

⚠️ 중요 안내

계정 삭제 시 모든 개인 데이터가 영구적으로 삭제되며, 복구할 수 없습니다. 삭제 전 필요한 데이터는 미리 백업해주세요.

삭제되는 데이터 목록

  • 계정 정보: 사용자 프로필, 로그인 정보
  • 학습 데이터: 생성한 단어장, 학습 기록, 진도
  • 설정 정보: 앱 설정, 알림 설정
  • 팟캐스트 기록: 생성한 팟캐스트, 재생 기록
  • 퀴즈 결과: 퀴즈 점수, 성취도 기록
  • 클라우드 동기화 데이터: 클라우드에 저장된 모든 데이터

삭제 처리 시간

계정 삭제 요청 후 즉시 계정 및 관련 데이터가 삭제됩니다. 일부 백업 시스템에서는 최대 30일 이내에 완전히 제거됩니다.

앱 삭제 vs 계정 삭제

앱 삭제 (언인스톨)

• 기기에서만 앱 제거
• 계정 데이터는 서버에 보존
• 재설치 시 데이터 복원 가능

계정 삭제

• 모든 서버 데이터 완전 삭제
• 복구 불가능
• 새로 가입 시 처음부터 시작
